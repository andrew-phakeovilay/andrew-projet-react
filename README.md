# Projet React + TypeScript + Vite + Tailwind

API comparaison de prix des jeux PC digitals : [Cheapshark](https://apidocs.cheapshark.com/)

- DEMO du site : (https://andrew-phakeovilay.github.io/andrew-projet-react/)

## Installation

```
> npm install
> npm run dev
```

## Liste des fonctionnalités effectuées

- Liste des jeux / jeux favoris
- Recherche de jeu
- Ajout de mettre / enlever les jeux favoris
- Tri des jeux / jeux favoris par nom ou prix
- Détail d'un jeu
- Thème dark / light

(j'espère avoir rien oublié)
